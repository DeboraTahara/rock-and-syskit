Notice that in Syskit two components that are connected to each other do not have a dependency relationship. This is a critical point in component-based system design that is often understood wrong. A component does not depend on its sources to be functioning or faulty. The network that binds them together is. This is a much richer representation of dependency, as it allows to isolate the role of each component in an overall function, and to build either degraded or orthogonal functions on the same set components.
